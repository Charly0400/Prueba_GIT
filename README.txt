# PRUEBA UNITY GIT

Este es un proyecto para enseñar la utilización de git y herramientas

## INTALACIÓN

Para poder utilizar este proyecto se debe tener instalado Unity 2021.2.5

Para descargar este proyecto, utiliza el comando

``` bash
$ git clone https://github.com/Charly0400/Prueba_GIT.git
```
